# Monte Sinai Ecommerce

Plataforma completa de ecommerce para venda de água, gás e produtos de limpeza, desenvolvida com HTML, CSS, JavaScript, Supabase e Netlify.

## Destaques do Projeto

✅ Autenticação de usuários

✅ Controle de permissões (RBAC)

✅ Painel administrativo completo

✅ Gestão de produtos

✅ Controle de estoque

✅ Gestão de pedidos

✅ Checkout integrado ao WhatsApp

✅ Upload de imagens

✅ Banco de dados PostgreSQL (Supabase)

✅ PWA instalável

✅ Deploy automatizado

✅ Responsividade Mobile, Tablet e Desktop

## Tecnologias

* HTML5
* CSS3
* JavaScript
* Supabase
* PostgreSQL
* Netlify
* GitHub
